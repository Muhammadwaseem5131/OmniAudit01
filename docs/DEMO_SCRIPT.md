# OmniAudit Demo Script

## Goal
Show how OmniAudit scans code, identifies security issues, and guides remediation in a short, compelling demo.

## Suggested length
60 to 90 seconds

## Script

1. Open the app and say: "OmniAudit is an AI-powered security auditing workflow for developers."
2. Upload the included demo project or the vulnerable sample folder. Explain that it uses safe placeholder credentials so judges can safely see how the scanner works.
3. Show the dashboard and point out the severity breakdown and security score.
4. Click one finding and explain what it means, such as a hardcoded API key or SQL injection pattern.
5. Switch to the remediation view and show the proposed fix.
6. Close by saying: "This demonstrates how AI agents can make security review faster, clearer, and more actionable."

## Tips
- Keep the narration short and focused.
- Highlight one or two findings clearly rather than showing everything at once.
- Use the sample app so the vulnerabilities are easy to understand.
